隐私政策

本隐私政策适用于 Chrome 扩展程序「天音 AI 创作助手」（以下简称"本扩展"）。

最后更新日期：2026 年 5 月 15 日

1. 我们收集哪些数据

本扩展仅在用户主动操作时处理以下数据：

  • DeepSeek API Key：由用户自行填写，仅存储在浏览器本地（chrome.storage.local），不会上传至任何服务器。
  • 歌词 / 创作内容：当用户点击"润色""重写""续写"等按钮时，输入框中的文字会被发送至 DeepSeek API 以生成 AI 建议。此数据由用户主动触发传输，本扩展不做任何存储或二次使用。

本扩展不收集以下信息：个人身份信息、健康信息、财务信息、位置信息、浏览历史、点击记录或任何其他与创作功能无关的数据。

2. 数据如何使用

  • API Key 仅用于向 DeepSeek API 进行身份验证，不用于任何其他目的。
  • 创作内容仅在用户触发 AI 功能时临时传输至 DeepSeek API，用于生成建议文本，传输完成后本扩展不保留副本。

3. 第三方服务

本扩展调用 DeepSeek 开放平台 API 实现 AI 功能。用户的创作内容将依据 DeepSeek 的服务条款和隐私政策进行处理，请参阅：https://platform.deepseek.com/privacy_policy。

本扩展不与任何其他第三方共享用户数据。

4. 数据存储与安全

  • API Key 存储于 Chrome 浏览器本地存储，受浏览器沙箱保护，仅本扩展可读取。
  • 所有 API 请求通过 Service Worker 代理转发，不暴露 API Key 给页面脚本。
  • 本扩展不设服务器端，无数据库，不存在数据泄露至开发者的渠道。

5. 用户权利

用户可随时在扩展设置面板中修改或清除 API Key。卸载本扩展后，所有本地存储数据将被浏览器自动清除。

6. 适用范围

本扩展仅在用户访问 https://music.163.com/st/tianyin/ 相关页面时激活，不在其他网站运行。

7. 政策变更

如本政策发生重大变更，我们将在 GitHub 仓库中更新本文件，并更新页面顶部的"最后更新日期"。

8. 联系方式

如有疑问，请通过 GitHub Issues 联系：https://github.com/chenpipi0807/ai-roundtable-extension/issues
